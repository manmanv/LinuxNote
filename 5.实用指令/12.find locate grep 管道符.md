#### find指令

find指令将从指定目录向下递归的遍历其各个子目录，将满足条件的文件或者目录显示在终端

###### 基本语法：

find 搜索范围 选项

选项说明：

| 选项            | 功能                               |
| --------------- | ---------------------------------- |
| -name<查询方法> | 按照指定的文件名查找模式或查找文件 |
| -user<用户名>   | 查找属于指定用户名所有文件         |
| -size<文件大小> | 按照指定的文件大小查找文件         |
